Como surgiu a RGS Dream Mobile?

A RGS Dream Mobile foi criada com o intuito de gerenciar os app's desenvolvidos por eu mes­mo Ricardo Gomes. No momento desenvolvi 2 aplicativos, um para compartilhamento de gifs que se chama Spread Gifs e outro voltado para auxílio no ensino de matemática. O app vol­ta­do para ensino se chama Exemplifica e conta com conteúdo próprio desenvolvido es­pe­ci­al­men­te para uma compreensão simples e direta de cada assunto abordado, podendo se tornar uma ferramenta muito útil principalmente para adolescentes que tem dificuldades com ma­te­má­ti­ca. O Exemplifica também conta com calculadoras específicas, também desenvolvidas es­pe­ci­al­men­te para auxílio em cada área.
